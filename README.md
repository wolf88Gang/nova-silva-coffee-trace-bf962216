# Nova Silva Platform

Plataforma privada de gestión B2B para cadenas de valor agrícola.

## Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (auth, database, edge functions)

## Setup local

```sh
git clone <REPO_URL>
cd nova-silva
npm i
npm run dev
```

## Despliegue

El frontend se publica vía CI/CD. Las edge functions y migraciones de base de datos se despliegan automáticamente.
